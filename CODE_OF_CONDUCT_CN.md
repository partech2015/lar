# 行为准则 — Lár

我们致力于将 *Lár 引擎* 社区打造为一个充满尊重、无骚扰的体验环境，无论参与者的背景如何。本行为准则适用于所有项目空间，包括 GitHub 仓库、讨论论坛和社区活动。

我们采用并执行 **贡献者公约 (Contributor Covenant)** 的原则，将其作为社区标准的基线。

## **我们的誓言**
我们承诺通过能为开放、包容、多样化和健康的社区做出贡献的方式进行行动和互动。这包括保持专业精神、培育**建设性的对话**，并将成员的**福祉**放在首位。

## **我们的行为标准**

在所有社区互动中，以下行为是预期且强制性的：

- **友善与尊重**：以同理心、友善和尊重的态度对待他人。技术观点的差异是正常的，但人身攻击和侵略性行为是不可接受的。

- **建设性反馈**：将反馈集中在代码和创意上，而不是个人身上。提供有助于改进项目的深思熟虑的批评。

- **尊重隐私和界限**：尊重保密性，未经明确允许，不得分享私人信息。

- **避免骚扰和歧视**：我们禁止任何形式的骚扰，包括与性别身份、性取向、残疾、身体外观、体型、种族、宗教或技术能力相关的攻击性言谈。

## **报告与执行**

**A. 举报违规行为**

如果您目睹或遭受了违反本行为准则的行为，请及时且机密地向项目维护者举报：

- **电子邮件**：[axdithya@snath.ai](mailto:axdithya@snath.ai), [vinay@snath.ai](mailto:vinay@snath.ai)

- **举报内容**：请包括必要的背景信息、涉及人员姓名以及任何证据（截图、链接），以帮助维护者高效地调查情况。

**B. 执行措施**

所有报告都将得到审查、调查，并得到谨慎且公正的处理。项目维护者负责阐明可接受行为的标准，并将针对任何违规行为采取迅速且适当的纠正措施。

违规的后果可能包括：

- **警告**：一份私下的书面警告。

- **临时禁言**：临时排除在项目空间之外。

- **永久禁言**：在持续不当行为的情况下，永久从所有项目空间中移除。
